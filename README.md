# Hyper Serious Circle Game Made For A Very Serious Game Jam

Have YOU ever wanted to click a circle. Well now you can, professionally. Get the highest score in the worlds most professional (aka garbage) high score game.

Professionally I would recommend downloading then playing the game, our tight budget made the web build 'professionally lacking'

Itch Page: https://zirosi.itch.io/hscgmfavsgj

Github Page: https://github.com/zirosi/hyper-serious-circle-game


## Controls

This is a one button game, you can either use the "w" key or just click with left click



## Install Instructions (*Unneeded*)

If you are on a desktop computer the x86_64 should work, but if you have an Arm64 cpu (mainly on new laptops) you will need to download the Arm64 variant. 

*WARNING* The Arm64 builds have not been tested due to a lack of funding, please comment any issues with the Arm or x86_64 versions, sorry for the inconvenience
